# RM Tools 1.2.0

Клиентский мод для персонала RaidMine на **Minecraft 1.21.11 / Fabric / LabyMod 4**.

## Возможности

- ЛКМ по точному нику в чате открывает меню наказаний. Префиксы и служебные слова `server`, `chattools`, `cmi` игнорируются.
- Ник под курсором подсвечивается оранжевым градиентом и мягким свечением.
- Современный интерфейс без низкоразрешённых bitmap-иконок: значки и шрифт генерируются со сглаживанием.
- Компактная верхняя панель: баны, муты, варны и время сессии.
- Right Shift открывает редактор HUD; повторный Right Shift в редакторе центрирует панель.
- Автоматический скриншот после наказания: `Ник_ПунктПравил_Дата.png`.
- В серверную команду по умолчанию передаётся только пункт правила, например `2.2`.
- Настройки через шестерёнку: цвет интерфейса, формат причины, автоскриншоты, уведомления и список запрещённых слов.
- Запрещённые слова подсвечиваются в чате; верхняя панель плавно сменяется уведомлением о нарушении.
- При упоминании вашего ника появляется уведомление над хотбаром.

## Горячие клавиши

- **ЛКМ по нику** — открыть меню наказаний.
- **`/rmp Ник`** — резервное открытие меню.
- **Right Shift** — редактор HUD.
- В редакторе: Right Shift или `C` — центрировать, `R` — сбросить, колесо — масштаб, Esc — сохранить.

## Сборка

Требуется Java 21. GitHub Actions уже настроен в `.github/workflows/main.yml`.

```bash
gradle --no-daemon clean build
```

Готовый файл:

```text
build/libs/rm-tools-1.2.0.jar
```

## Конфигурация

После первого запуска создаётся `config/rm-tools.json`. Старый `config/raidmine-staff.json` переносится автоматически.

Перед применением проверьте синтаксис команд RaidMine в конфиге. По умолчанию мод отправляет команды только на сервере, адрес которого содержит `raidmine`.
