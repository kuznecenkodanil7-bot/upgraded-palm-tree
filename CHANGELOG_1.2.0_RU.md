# RM Tools 1.2.0

- Название изменено на RM Tools.
- Установлен предоставленный логотип RM; добавлено мягкое свечение в HUD.
- Полностью обновлены иконки и сглаженный интерфейсный шрифт.
- Исправлено определение ника по точному слову под курсором и списку игроков.
- Исключены ложные значения `server`, `chattools`, `cmi` и клики по префиксам.
- Добавлена подсветка ника оранжевым градиентом при наведении.
- В HUD кики заменены на варны.
- Right Shift в редакторе центрирует панель.
- Добавлены автоматические скриншоты после наказания.
- Причина команды по умолчанию содержит только пункт правила.
- Добавлены настройки цвета, причины, уведомлений и запрещённых слов.
- Запрещённые слова подсвечиваются; HUD временно показывает уведомление о нарушении.
- Добавлено уведомление над хотбаром при упоминании вашего ника.
