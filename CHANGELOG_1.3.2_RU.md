# RM Tools 1.3.2

- Исправлено отображение `net.minecraft.client.user` вместо игрового ника.
- Логин теперь автоматически определяется по текущему игровому профилю.
- До подключения к серверу используется `MinecraftClient#getSession().getUsername()`.
- После подключения приоритет имеет фактический ник `ClientPlayerEntity#getGameProfile().name()`.
- Пользователь вводит только пароль; поле логина не требуется.
