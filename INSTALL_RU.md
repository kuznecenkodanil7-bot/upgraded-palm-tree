# Установка RM Tools в LabyMod 4

## Требования

- Minecraft 1.21.11
- LabyMod 4 с профилем Fabric
- Java 21
- собранный `rm-tools-1.2.0.jar`

## Установка

1. Соберите проект через GitHub Actions или командой `gradle --no-daemon clean build`.
2. Возьмите `build/libs/rm-tools-1.2.0.jar`.
3. Откройте папку выбранной Fabric-версии LabyMod 4.
4. Поместите JAR в папку `mods`.
5. Запустите игру через тот же профиль Minecraft 1.21.11.
6. Подключитесь к RaidMine и откройте чат.

## Проверка

- Нажмите ЛКМ именно по буквам ника игрока. Откроется меню RM Tools.
- Резервная команда: `/rmp НикИгрока`.
- Нажмите Right Shift — откроется редактор верхней панели.
- В редакторе нажмите Right Shift ещё раз, чтобы центрировать HUD.
- Настройки открываются шестерёнкой в правом нижнем углу меню наказаний.

## Скриншоты

После успешной отправки наказания скриншот автоматически сохраняется в папку Minecraft `screenshots` с именем вида:

```text
PlayerName_2.2_2026-07-15_23.40.00.png
```

## Конфиг

Файл: `config/rm-tools.json`.

Проверьте шаблоны `warnCommand`, `muteCommand`, `banCommand` и `permanentBanCommand`. Значение `punishmentReasonTemplate` по умолчанию равно `{rule}`, поэтому сервер получает только пункт правила.
